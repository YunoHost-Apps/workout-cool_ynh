This package is not well finished for self hosting.

There is currently no way to enable all features that normally are for paid users. It’s [reported](https://github.com/Snouzy/workout-cool/issues/177) but with no result for now...

No workout database is integrated in the app, you have to upload yours. For more info you can look [here](https://github.com/Snouzy/workout-cool?tab=readme-ov-file#exercise-database-import).
